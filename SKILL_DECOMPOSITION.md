# Skill-Decomposition — Euterpe

Lebendes Arbeitsdokument. Wir zerlegen den obersten Skill so lange, bis jedes
Blatt **atomar** ist.

## Methode

1. **Atomar** = jemand ohne jede Vorkenntnis kann *diesen* (Sub-Sub-…)Skill direkt
   lernen.
2. **Zerlegungs-Regel:** erst **zeitlich** zerlegen (die Handlung in ihre
   zeitlichen Einzelteile). Geht das nicht → nach **Skill-Komponenten** zerlegen
   (was muss ich können, um das zu können?).

## Legende

- ✓ **atomar** — von null lernbar, ein Lernziel-Blatt
- ⧖ **noch zu zerlegen**
- ⊕ **Vorbedingung/Zustand** — außerhalb des Lernziels (bewusst nicht zerlegt)
- 🎮 hat schon eine spielbare Challenge

---

## AKTIVES ZIEL — Pareto-Unterziel (20/80)

> **Eine Melodie, die ich schon spielen kann, links mit einfachen Akkorden
> (I/IV/V) im 4/4 auf 1-2-3-4 begleiten — beide Hände zusammen, klingt ordentlich.**

Bewusst die 20 % Fertigkeiten für 80 % Spaß. Die **Melodie ist hier *Input***
(geübt, z. B. eine Zelda-Melodie), nicht Lernziel. Der Ohr-Transkriptions-Strang
(Schritt 1 unten) fällt damit vom kritischen Pfad — er gehört zum Nordstern.

> Diese Schärfung löst zwei frühere Lücken auf: **Melodie-Rhythmus** steckt im
> „kann ich die Melodie schon", **Improvisation** entfällt (Melodie ist fix).

### Die 20 % (kritischer Pfad)

1. **Akkorde bereitstellen**
   - 3 Hauptakkorde (I/IV/V) **greifen** → ✓ haben (Akkordgriff `w2` 🎮)
   - Tonart finden: *Grundton + Dur/Moll* (klein) — oder nachschlagen
2. **Pro Takt Akkord wählen**
   - **„Passt / passt nicht" hören** → ⭐ neues Blatt, der eine Kern-Ohr-Skill
   - 1× pro Takt wechseln (minimal); Akkordfolge nachschlagen als Stütze erlaubt
3. **Im 4/4 anschlagen**
   - *Puls fühlen + auf 1-2-3-4 timen* → neues Blatt
   - *Akkord rechtzeitig wechseln* → neues Blatt
4. **Hände zusammen** (RH bekannte Melodie + LH Block-Akkorde)
   - LH automatisieren + langsam auf dem Puls zusammenführen → **die einzige echte Wand**

### Neu zu bauen (nur 4 kleine Blätter)

*„passt"-hören* · *Puls/4-4-Timing* · *Akkordwechsel-Motorik* · *Hände-zusammenführen*
**Schon da:** Akkordgriff, Tastenfinder. **Off-path (erst Nordstern):** Hörtrainer (Richtung hören).

### Voraussetzung (Input, nicht Lernziel)

Eine Melodie rechts spielen können (geübt). *Nordstern-Upgrade:* sie nach Gehör finden.

---

## Nordstern-Ziel (Referenz, später)

> **Aus einer inneren Melodie ein vollständiges Klavierstück spielen.**
> (Die Melodie darf sich beim Spielen weiterentwickeln. Fokus liegt auf dem
> *Spielen*, nicht auf Notieren.)

*Die folgende vollständige Zerlegung bleibt als Landkarte fürs große Ziel stehen.
Fürs aktive Pareto-Unterziel ist nur ein Ausschnitt davon kritisch (s. o.).*

⊕ *Vorbedingung (nicht Teil des Ziels):* eine Melodie im Kopf haben & halten
(inneres Hören). Notieren-Können wäre ein eigenes Problem.

### Zeitliche Zerlegung (Top-Kette)

1. **Mir die Noten der Melodie auf den Tasten zusammensuchen** ⧖ *(in Arbeit)*
2. Harmonie darunterlegen ⧖
3. Begleit-Textur & Form bauen ⧖
4. Melodie + Begleitung gleichzeitig spielen ⧖

---

## 1. „Noten zusammensuchen"

**Zeitlich** — aber als **Schleife pro Ton** (kein linearer Ablauf):

- a. Den nächsten Ton **innerlich hören** (Ziel) → ⊕ Vorbedingung
- b. Eine Taste **tippen** (Kandidat) → ✓ atomar (motorisch trivial)
- c. Kandidat mit Ziel **vergleichen** — passt / höher / tiefer? → ⧖ Gehör
- d. **Korrigieren**, bis es passt → braucht *Tastatur-Orientierung (rechts = höher)* → ✓ atomar
- e. Den Ton **behalten**, weiter zum nächsten → ⧖ *relationales Merken*

Bottomt also in drei Komponenten:
**Richtung/Abstand hören**, **Tastatur-Orientierung**, **relationales Merken**.

### 1e. „Relationales Merken"

Nicht zeitlich → **Skill-Komponenten**, als Tiefen-Leiter grob → fein.
(Bewusste Entscheidung: *relational* merken statt Notennamen-Liste oder rein
motorisch — weil transponierbar und gleiche Währung wie die Harmonie.)

- **Kontur erfassen** (hoch/runter/gleich) → ✓ **atomar** = *Richtung hören* 🎮 (Hörtrainer `g0`)
- **Schrittgröße erfassen** (Intervall) → siehe **1e-2**
- **Am Anker festmachen** (Stufe bzgl. Grundton) → siehe **1e-3**

> **Leiter, kein Entweder-oder:** Kontur + Intervall reichen zum Wiederfinden
> nach Gehör. *Stufen* ist das Upgrade, das mit der Harmonie (Schritt 2) verbindet.
> Reihenfolge: **Kontur → Intervall → (Grundton) → Stufen.**

#### 1e-2. „Intervalle hören"

Nicht zeitlich → Komponenten (Ankerlied-Trick):

- **Richtung hören** → ✓ haben (`g0`), Voraussetzung.
- **Ankerlieder im Gedächtnis haben** (Intervall ↔ bekannter Liedanfang) → ✓ **atomar** (Wissen, Auswendiglernen).
- **Gehörten Abstand mit dem Anker abgleichen** → ✓ **atomar (trainierbar)**, der eigentliche Übungs-Knoten.

#### 1e-3. „Stufen"

*Einem Ton seinen Platz bzgl. des Grundtons geben.* Nicht zeitlich → Komponenten:

- **Grundton heraushören** → ✓ **atomar (perzeptives Blatt)** — das „Nach-Hause-Kommen", den Ruheton finden/mitsingen; von null trainierbar.
- **Abstand Grundton → Ton hören** → = Intervall vom Grundton, baut auf **1e-2**, haben.
- **Dem Abstand einen Stufen-Namen geben** → ✓ **atomar** (Wissens-Zuordnung).

---

## 2. „Harmonie darunterlegen"

Beide Wege **verschränkt** (hörend + ableitend). **Zeitlich:** einmaliges Setup
+ Schleife pro Abschnitt.

**Setup (einmal) — Tonart/Grundton bestimmen** (legt die Akkord-Palette fest):
- *Grundton heraushören* → ✓ haben (`1e-3`)
- *Dur/Moll hören* (Geschlecht) → ✓ **atomar** (neues perzeptives Blatt)
- *Welche Akkorde zur Tonart gehören* (I/IV/V) → ✓ **atomar** (Wissen)

**Schleife pro Abschnitt:**
- a. **Akkordwechsel hören** (wo ist ein Wechsel fällig?) → ✓ **atomar (perzeptiv)**
- b. **Tragenden Akkord bestimmen** — zwei verschränkte Wege:
  - *Hörend:* Kandidaten unterlegen + vergleichen → *Akkord greifen* (✓ `w2` 🎮) + *„passt"-beurteilen* → ✓ **atomar (perzeptiv)**
  - *Ableitend:* Melodieton → Stufe (✓ `1e-3`) → welcher Hauptakkord enthält die Stufe? → ✓ **atomar** (Wissen)
  - Praxis: ableiten grenzt ein (1–2 Kandidaten), das Ohr entscheidet.
- c. **Akkord greifen** → ✓ haben (`w2`, Akkordgriff 🎮)
- d. **Prüfen & korrigieren** → nutzt *„passt"-beurteilen* (s. b)

> Bottomt fast komplett in **schon Vorhandenes** + **3 neue atomare Blätter**:
> *Dur/Moll hören*, *Akkordwechsel hören*, *„passt"-beurteilen*.

---

## 3. „Begleit-Textur bauen" (Minimal-zuerst)

Entscheidung: **schnellster Pfad** → Textur statt voller Form; innerhalb der
Textur die *minimale* Variante (Akkord im Puls anschlagen) zuerst.

**Zeitlich** — Schleife pro Schlag:
- a. Akkord **greifen** → ✓ haben (`w2`, Akkordgriff 🎮)
- b. Im **gleichmäßigen Puls anschlagen** → siehe Puls
- c. **Rechtzeitig zum nächsten Akkord wechseln** (Griff vorbereiten) → ✓ **atomar (motorisch)**

**„Im gleichmäßigen Puls spielen"** (`p1`):
- *Puls fühlen/halten* (innere Uhr, mitklopfen) → ✓ **atomar (perzeptiv/motorisch)**
- *Aktion auf den Schlag timen* → ✓ **atomar (motorisch)**

→ Bottomt in **Akkord greifen (haben)** + 3 atomare Blätter:
*Puls fühlen*, *auf den Schlag timen*, *Griff rechtzeitig wechseln*.

### Kür — aufgeschoben (nicht auf dem schnellen Pfad)

- **Muster-Repertoire** (gebrochen / Bass + Akkord / pumpen) — Upgrade, kein Blocker.
- **Form**: fürs Ziel reicht *sauber anfangen + auf dem Grundton-Akkord enden*.
  Intro/Steigerung/Arrangement bewusst später.

---

## 4. „Melodie + Begleitung gleichzeitig spielen"

**Komponenten** (gleichzeitig, nicht zeitlich). Minimal: rechts Melodie, links
schlichte Akkorde im Puls.

- **Rechts: gefundene Melodie spielen** → ✓ haben (Schritt 1 + Motorik)
- **Links: Begleitung spielen** → ✓ haben (Schritt 3)
- **Beide Hände koordinieren** ← das einzig Neue (`m3`):
  - *Gemeinsamer Puls als Anker* → ✓ haben (`p1`)
  - *Jede Hand einzeln automatisieren* → ✓ **atomar (motorisch)**
  - *Langsam zusammenführen, an gemeinsamen Schlägen ausrichten* → ✓ **atomar (Koordination)**

**Kür (aufgeschoben):** echte Unabhängigkeit (links anderer Rhythmus als rechts).

## Minimaler Pfad zum Ziel (schnellste Linie)

Tasten finden → Richtung → Intervalle → *(Melodie wiederfinden)* → Grundton + Dur/Moll
→ Akkorde der Tonart + Stufe→Akkord → Akkord greifen + „passt"-hören
→ Puls + Akkord im Puls + Wechsel → Hände zusammenführen → **Ziel**.

*Bewusst ausgelassen (Kür):* feine Intervall-Präzision, Begleit-Muster-Repertoire,
Form/Arrangement, volle Hand-Unabhängigkeit.
